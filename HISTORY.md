# Public Project History

Actanara completed its development before version 1.0.0 in a private
repository. That private repository remains the authority for pre-release
development history, internal audit records, and historical evidence.

The public Git history begins with Actanara v1.0.0 on 2026-07-13. Earlier
private commits and tags are intentionally not imported, reconstructed, or
backdated. Public development after v1.0.0 will proceed through ordinary
forward-only commits and immutable release tags in this repository.

The immutable v1.0.0 public release was withdrawn after post-publication E2E
found that an upgrade could preserve stale concrete source paths in managed
LaunchAgents. v1.0.1 is the forward-only corrective release; the v1.0.0 tag,
artifacts, and root commit remain unchanged as public audit evidence.
