# Open Nova Dashboard 真实静态 Demo

本目录直接使用 Open Nova 真实 Dashboard 的 HTML、CSS、组件、布局与交互代码；没有重新设计或另画界面。静态数据层负责拦截后端 API，使页面可离线审核。

## 本次保留内容

- 当日实时总览
- AI 资产
- Nova-Task 任务看板
- 一份周报：`2026-W27`（2026-06-29 ～ 2026-07-05）
- 两份普通日记：`2026-07-03`、`2026-07-05`
- 一份 Blank Day 日记：`2026-07-04`

Foundation 运维、nova-RAG、月报、设置及其他日期的数据未纳入这份公开演示快照。

## 本地查看

直接打开 `index.html`，或在本目录启动任意静态文件服务器。页面资源均使用相对路径；图表与 Markdown 依赖已随目录保存。

`tasks.html` 是同一份真实 Dashboard 中的 Nova-Task 页面。演示中的写操作只修改当前浏览器内存，不会写回 Open Nova 数据。

## 数据说明

- 总览、AI 资产与任务看板来自既有静态快照。
- 三份日记来自本机 Open Nova Dashboard 的只读 API 快照。
- W27 周报的页面结构与字段契约来自真实 Dashboard；为完整展示 KPI、趋势、模型、工作区、知识资产、主题和经验卡片，指标已依据既有 W27 截图口径补入展示性静态数据，不代表某次真实运行统计。
- `2026-07-04` 的 `activityState` 为 `empty`，用于展示真实 Blank Day 渲染。
- AI 资产的基础设施仅保留 Mac mini 与华硕路由器；本机绝对路径统一显示为 `~` 开头的相对表达。
- Skill 库仅保留名称、描述、分类和路径等展示元数据，不打包 Skill 正文或全文搜索快照。
